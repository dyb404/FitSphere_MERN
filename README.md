 # FitSphere — Full-Stack Fitness Management Platform

FitSphere is a MERN-stack web application that connects clients, trainers, and admins in one seamless fitness ecosystem.

JWT-based authentication with role-based access control across three user roles: Admin, Trainer, and Client.
Trainers create and assign personalized workouts and diet plans directly to their clients.
Clients track progress logs including weight, calories, and notes through a dedicated dashboard.
Class scheduling, health tips, and diet plan management for a complete wellness experience.
RESTful Express API with 9 modular route handlers backed by MongoDB via Mongoose.
React + Vite frontend using Context API for global auth state and protected route navigation.
Built with: MongoDB, Express, React, Node.js, JWT, bcryptjs, React Router v6.
